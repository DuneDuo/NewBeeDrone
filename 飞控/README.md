## 概述

本仓库收录 DM-FC01 控制板对外资料。目录按客户查阅顺序保留真实资料，空占位目录不再保留。

## 资料目录

- [说明书/](说明书/) - 使用说明、数据手册和接口文档
- [固件/](固件/) - 可直接使用或烧录的固件文件
- [3D模型/](3D模型/) - STEP、STP、SolidWorks 或结构压缩包等 3D 模型资料

## 维护规则

- 新增资料必须放入对应中文目录，不新增空目录。
- 2D 图纸和 3D 模型直接放在根目录下的 2D图纸/、3D模型/。
- 代码工程、第三方工程和已有压缩包保持原文件名，避免破坏客户使用路径。
