> 详见记忆索引：[[MEMORY]]、[[project-overview]]、[[decisions-log]]

## 项目简介

基于 ArUco 码视觉融合定位与自动换电的封闭园区全自动无人机投递终端与系统开发。按四层架构组织：

```
  服务层  — 云服务器 + 数据库 + 业务逻辑（阿里云 ECS）
  通讯层  — 4G 链路 + MAVLink 透传 + 视频推流（EC20 ×2）
  电站层  — 换电 + 充电 + 投放舱 + 降落伞（地面端硬件）
  飞机层  — 飞控 + 树莓派 + ArUco + 动力（空中端硬件）
```
详见：`通信链路与MAVLink协议学习笔记.md`

## 关键决策

- 固件：ArduPilot 为主，社区大、资料多
- 策略：3B 跑通逻辑 → 性能摸底 → 升级平台
- 数据库：前期 SQLite，后期切 InfluxDB
- 机器视觉：飞机端跑，不走云端
- 自动换电：先做快拆充电，再做全自动
- 4G 断连：飞控 failsafe 必须配好
- 服务器：免费试用 3 个月 → 科研算力包 ¥2000 额度续
